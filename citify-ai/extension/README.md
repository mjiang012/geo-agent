# Citify AI Browser Extension

Browser extension for real-time content optimization and analysis.

## Features

- Real-time page analysis
- RAG friendliness score
- Optimization suggestions
- Quick access to Citify AI dashboard
- Configuration options

## Installation

### Development

1. Open Chrome/Edge and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `extension` directory

### Production

1. Build the extension: `npm run build`
2. Install from the `dist` directory

## Configuration

1. Right-click the extension icon and select "Options"
2. Configure API URL and other settings
3. Save your changes

## Usage

1. Navigate to any webpage
2. Click the Citify AI extension icon
3. Click "Analyze Page" to get recommendations
4. Use the suggestions to optimize your content
